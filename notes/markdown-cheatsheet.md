# Markdown Cheatsheet

A quick reference for Markdown syntax supported by Notewall.

---

## Text Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strike~~` | ~~strike~~ |
| `` `code` `` | `code` |

## Headers

```markdown
# H1
## H2
### H3
#### H4
```

## Links & Images

```markdown
[Link text](https://example.com)
![Alt text](image.jpg)
```

## Lists

```markdown
- Unordered item
- Another item
  - Nested item

1. Ordered item
2. Another item
```

## Code Blocks

Use triple backticks with language name:

~~~markdown
```python
print("Hello!")
```
~~~

## Blockquotes

```markdown
> This is a quote
> It can span multiple lines
```

## Horizontal Rule

```markdown
---
```

## Tables

```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
```
